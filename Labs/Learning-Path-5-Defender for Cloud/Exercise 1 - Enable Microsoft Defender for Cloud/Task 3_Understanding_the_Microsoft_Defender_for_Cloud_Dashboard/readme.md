# Taller 3 — Understanding the Microsoft Defender for Cloud Dashboard

## Descripción
En este laboratorio se exploró el panel **Overview de Microsoft Defender for Cloud** con el objetivo de comprender cómo se visualiza la postura de seguridad de una suscripción en Azure.

Se revisaron los principales pilares de seguridad, el estado de cumplimiento y la información general que proporciona el dashboard para el monitoreo continuo.

---

## Objetivo del laboratorio

Comprender la estructura y los componentes del panel de Microsoft Defender for Cloud, identificando la información clave sobre postura de seguridad, cumplimiento y protección de cargas de trabajo.

---

## Acceso al panel

1. Ingresar al portal de Azure: https://portal.azure.com  
2. Buscar **Defender** en la barra de búsqueda  
3. Seleccionar **Microsoft Defender for Cloud**  
4. En el menú izquierdo abrir **Overview**

---

## Exploración del Dashboard

El panel Overview proporciona una vista unificada de la postura de seguridad y contiene los siguientes pilares:

- Security posture  
- Regulatory compliance  
- Workload protections  
- Inventory  
- Network / Data security  

Cada uno permite acceder a información detallada sobre el estado de seguridad de la suscripción.


---

## Security Posture (Secure Score)

En esta sección se visualiza:

- Puntuación de seguridad (Secure Score)  
- Controles completados  
- Recomendaciones  

En entornos de laboratorio el puntaje puede aparecer como **N/A**, ya que el cálculo puede tardar hasta 24 horas.

---

## Regulatory Compliance

El panel muestra el estado de cumplimiento basado en estándares de seguridad.

En este laboratorio no se observaron evaluaciones activas, lo cual es normal en suscripciones nuevas sin políticas aplicadas.

---

## Workload Protections

Se verificó la cobertura de protección de recursos y la ausencia de alertas activas, confirmando que Microsoft Defender for Cloud se encuentra monitoreando la suscripción.

---

## Inventory

Se revisó el inventario de recursos evaluados por Defender for Cloud, confirmando que los recursos de la suscripción están siendo analizados.
